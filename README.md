# 区块链系统Demo

## 如何运行

1. 从 [NodeJs Official Page](https://nodejs.org/en/download/)下载并安装`NodeJs LTS` 版本
2. 导航到根目录，运行`npm install`安装项目依赖
3. 运行`npm run serve`运行项目
